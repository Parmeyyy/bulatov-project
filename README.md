# Эрик Булатов. Избранные страницы

Зачётный проект по культурологии — сайт-репортаж о посещении выставки.

**Студент:** Евстифеев Глеб  
**Группа:** 24ИЖ1  
**Дата посещения:** 17 мая 2026 года  
**Место:** РОСИЗО, Петроверигский переулок, Москва

---

## О проекте

Сайт сделан как визуальный отчёт о первой посмертной выставке художника-концептуалиста Эрика Булатова «Избранные страницы» в РОСИЗО. Внутри — личные фотографии с выставки, текст эссе и разбор философии Булатова: конфликт плоскости и глубины, советский плакат как клетка, небо как выход.

Булатов умер в ноябре 2025 года в Париже, прожив 92 года. Эта выставка — первая после его смерти.
